1. 标题

#代表标题格式，#越多即标题级别越小。目前Markdown共有六级标题，如下图所示（标题大小+对应快捷键）

用法 快捷键 示例
# 一级标题 ctrl+1
## 二级标题 ctrl+2
### 三级标题 ctrl+3
#### 四级标题 ctrl+4
##### 五级标题 ctrl+5
###### 六级标题 ctrl+6

<h1> 标题1</h1>

4. 基本格式

格式 使用方法 示例
加粗 ctrl+B/** markdown
斜体 ctrl+I/* markdown
下滑线 ctrl+U markdown
删除线 alt+shift+5/~~ markdown
代码 ctrl+shift+~ markdown 代码内含有单引号`单词`使用双引号
分隔线 ***/---/___ 见表格下（最好在分隔线前后添加空白行）
链接(邮箱) [显示文本](链接网址)/<链接网址>/name@user.com 这是小沪的博客链接 http://xiaohuge.icu

转义字符 \ 可使用的转义字符：` * _ {} [] ! # + | () .

如需同时使用粗斜体即***markdown
